# Hipoteca Financial Auditor

## Sistema Profesional de Auditoría Financiera de Créditos Hipotecarios

Este proyecto tiene como objetivo realizar una auditoría financiera integral de créditos hipotecarios mediante técnicas de matemática financiera, análisis económico y pericia contable.

## Funcionalidades

- Reconstrucción del cronograma de pagos.
- Cálculo de TEA y TEM.
- Auditoría de intereses.
- Comparación con tasas del mercado.
- Análisis de costo financiero.
- Indicadores financieros.
- Generación automática de gráficos.
- Elaboración de informe pericial.
